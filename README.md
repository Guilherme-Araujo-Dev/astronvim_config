# astonvim_config

Esse é o arquivo de configuração do AstroNVIM personalizado meu, baseado no [AstroVim]

Para instalar, segue as instruções abaixo:

1 - Faça o clone na pasta .config

```bash
cd ~/.config
git clone
```

2 - Renomeie a pasta para nvim

```bash
mv astonvim_config nvim
```

3 - Abra o nvim e instale os plugins

```bash
nvim
:PlugInstall
```

4 - Pronto, agora você tem o AstroVim personalizado por mim.
